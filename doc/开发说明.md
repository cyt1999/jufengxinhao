# 飓风信号网站 - Demo 开发说明

## 已完成的功能模块

### ✅ 基础架构
- [x] 全局样式配置（引入 theme.css）
- [x] 基础组件（Header, Footer, Button, Card）
- [x] 布局组件（统一布局结构）

### ✅ 页面模块
1. **首页** (`/`)
   - Hero区域（品牌展示）
   - 功能特色展示
   - 品牌故事概览
   - 职业发展路径
   - 统计数据展示

2. **关于我们** (`/about`)
   - 我们的故事
   - 我们的愿景
   - 团队路径展示

3. **实战训练营** (`/training`)
   - 核心价值展示
   - 流程说明
   - 申请方式

4. **博客** (`/blog`)
   - 文章列表展示
   - 文章卡片布局

5. **测评** (`/assessment`)
   - 心理测评功能
   - 20道题目
   - 5个维度评估
   - 结果展示

6. **FAQ** (`/faq`)
   - 常见问题分类
   - 手风琴式展开/收起
   - 5个分类，每个分类2个问题

7. **捐赠** (`/donate`)
   - 捐赠说明
   - 支付方式展示（线下支付）
   - 支付流程说明

## 技术栈

- **框架**: Next.js 16
- **样式**: CSS Variables + Tailwind CSS
- **语言**: TypeScript
- **内容管理**: JSON配置文件

## 文件结构

```
src/
├── app/
│   ├── layout.tsx          # 根布局
│   ├── page.tsx            # 首页
│   ├── about/
│   │   └── page.tsx        # 关于我们
│   ├── training/
│   │   └── page.tsx        # 实战训练营
│   ├── blog/
│   │   └── page.tsx        # 博客
│   ├── assessment/
│   │   └── page.tsx        # 测评
│   ├── faq/
│   │   └── page.tsx        # FAQ
│   └── donate/
│       └── page.tsx        # 捐赠
├── components/
│   ├── layout/
│   │   ├── Header.tsx      # 导航栏
│   │   └── Footer.tsx      # 页脚
│   └── ui/
│       ├── Button.tsx      # 按钮组件
│       └── Card.tsx        # 卡片组件
└── globals.css             # 全局样式

content/
├── zh/
│   ├── home.json           # 首页内容
│   └── assessment/
│       └── questions.json  # 测评题目

styles/
└── theme.css               # 配色方案
```

## 运行项目

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 启动生产服务器
pnpm start
```

## 注意事项

1. **配色方案**: 所有颜色定义在 `styles/theme.css` 中，修改此处即可全局生效
2. **内容管理**: 页面内容在 `content/zh/` 目录下的JSON文件中，方便更新
3. **组件复用**: 基础组件（Button, Card）可在各页面复用
4. **响应式设计**: 目前使用基础响应式布局，后续需要优化移动端体验

## 开发建议

1. 先运行项目，查看各个页面的效果
2. 根据实际效果，选择需要优先优化的功能
3. 逐步完善各个模块的功能和交互
4. 保持代码结构清晰，便于后续维护
