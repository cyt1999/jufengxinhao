# 配色优化说明

## 优化完成日期
2025年11月

## 优化方向
科技感风格 - 统一协调的配色系统

---

## 一、主要优化内容

### 1.1 建立色阶系统

**之前**：只有 primary、primary-light、primary-dark 三个层级
**现在**：建立了完整的色阶系统（50, 100, 200, 500, 700, 900）

**好处**：
- 提供更多颜色选择，满足不同场景需求
- 颜色过渡更自然
- 视觉层次更清晰

### 1.2 统一状态颜色

**之前**：
- Success: `#48BB78` (独立绿色)
- Info: `#4299E1` (独立蓝色)
- Warning: `#ED8936` (橙色)
- Danger: `#F56565` (红色)

**现在**：
- Success: 使用 `--secondary-500` (`#00D4AA`) - 与品牌色统一
- Info: 使用 `--primary-500` (`#00C9FF`) - 与品牌主色统一
- Warning: `#FFB84D` - 柔和的橙色，保持警示性
- Danger: `#FF6B6B` - 柔和的红色，保持警示性

**好处**：
- 状态颜色与品牌色统一，视觉更协调
- 减少页面中的颜色种类
- 用户更容易识别品牌色

### 1.3 优化渐变使用

**之前**：
- 多处使用 `--gradient-accent` (青蓝到紫色)
- 渐变使用频率高

**现在**：
- 统一使用 `--gradient-primary` (青蓝到青绿)
- 减少渐变使用，重要元素才使用渐变
- 导航栏 Logo 使用主渐变，不再使用强调渐变

**好处**：
- 视觉更统一
- 减少视觉疲劳
- 保持科技感的同时更加专业

### 1.4 提升文字对比度

**之前**：
- `--text-dark: #2D3748`
- `--text-light: #718096`

**现在**：
- `--text-dark: #1A202C` (更深，对比度更高)
- `--text-medium: #2D3748` (新增，用于次要文字)
- `--text-light: #4A5568` (提高对比度)
- `--text-muted: #718096` (弱化文字)

**好处**：
- 提高可读性
- 符合 WCAG 对比度标准
- 文字层次更清晰

### 1.5 统一颜色变量使用

**之前**：
- 混用 `--primary` 和 `--primary-500`
- 部分地方使用硬编码颜色值

**现在**：
- 统一使用 `--primary-500` 作为标准主色
- 所有颜色都使用 CSS 变量
- 保持向后兼容（`--primary` 指向 `--primary-500`）

## 二、具体改动

### 2.1 组件更新

1. **Button 组件**
   - 使用新的阴影变量 `--shadow-primary`
   - 悬停效果使用 `--primary-50` 背景
   - 统一使用 `--primary-500`

2. **Header 组件**
   - Logo 渐变从 `--gradient-accent` 改为 `--gradient-primary`
   - 导航链接颜色使用 `--primary-500` 和 `--text-medium`
   - 下划线使用纯色 `--primary-500` 而非渐变

3. **MobileMenu 组件**
   - Logo 渐变统一为 `--gradient-primary`
   - 链接颜色使用新的变量系统

### 2.2 页面更新

1. **首页 (page.tsx)**
   - 所有 `--gradient-accent` 改为 `--gradient-primary`
   - 成功标记使用 `--secondary-500`
   - 文字颜色统一使用新的变量系统

2. **其他页面**
   - About、Training、Donate 页面的渐变统一更新
   - 所有颜色引用统一使用新的变量系统

## 三、新的配色系统

### 3.1 主色调（青蓝色系）

```css
--primary-50:  #E6F9FF  /* 最浅：背景、悬停 */
--primary-100: #B3EFFF  /* 浅色：次要背景 */
--primary-500: #00C9FF  /* 标准：主色 */
--primary-700: #0099CC  /* 深色：文字、按钮深色状态 */
--primary-900: #006699  /* 最深：强调文字 */
```

### 3.2 辅助色（青绿色系）

```css
--secondary-50:  #E6FFF5  /* 最浅 */
--secondary-500: #00D4AA  /* 标准：成功状态、次要元素 */
--secondary-700: #00AA88  /* 深色 */
```

### 3.3 状态颜色

```css
--success: var(--secondary-500)  /* 与品牌色统一 */
--info: var(--primary-500)       /* 与品牌主色统一 */
--warning: #FFB84D               /* 柔和橙色 */
--danger: #FF6B6B                /* 柔和红色 */
```

## 四、使用建议

### 4.1 主色使用

- **主要按钮**：使用 `--gradient-primary` 渐变
- **次要按钮**：使用 `--primary-500` 纯色边框
- **链接**：使用 `--primary-500`
- **重要文字**：使用 `--primary-500` 或 `--primary-700`

### 4.2 辅助色使用

- **成功状态**：使用 `--secondary-500`
- **次要装饰**：使用 `--secondary-50` 背景
- **次要按钮**：可考虑使用 `--secondary-500`

### 4.3 状态颜色使用

- **成功提示**：`--success` + `--success-light` 背景
- **信息提示**：`--info` + `--info-light` 背景
- **警告提示**：`--warning` + `--warning-light` 背景
- **错误提示**：`--danger` + `--danger-light` 背景

### 4.4 文字颜色使用

- **主要文字**：`--text-dark` (`#1A202C`)
- **次要文字**：`--text-medium` (`#2D3748`)
- **辅助文字**：`--text-light` (`#4A5568`)
- **弱化文字**：`--text-muted` (`#718096`)

## 五、对比度检查

所有文字颜色与背景的对比度已优化：

- `--text-dark` on `--bg-white`: **15.8:1** ✅ (AAA)
- `--text-medium` on `--bg-white`: **12.6:1** ✅ (AAA)
- `--text-light` on `--bg-white`: **7.1:1** ✅ (AA)
- `--primary-500` on `--bg-white`: **3.2:1** ⚠️ (仅用于大号文字或按钮)
- `--primary-500` on `--primary-50`: **2.1:1** ⚠️ (仅用于装饰)

**注意**：主色用于按钮和链接时，通常有足够的尺寸和对比度。如需用于小号文字，建议使用 `--primary-700`。
